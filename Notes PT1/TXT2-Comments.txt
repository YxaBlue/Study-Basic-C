NOTES:
    >> COMMENTS
            - they are used for keeping track of your codes, aka DOCUMENTATION
            - the compiler ignores them.
            - They do nothing when you run your code

    >> You can use COMMENTS to:
        1. Explain what does this line/s of code mainly do.
        2. Documentation

    >> 2 Kinds of Comments:
        1. // Single Line Comment
        2. /* Multi-line Comment */
